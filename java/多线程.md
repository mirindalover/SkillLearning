
# 线程

## volatile关键字

![java内存模型][1]

	volatile之保证可见性(每个线程读取数据都是从同一主内存的共享变量中取，而不是从线程的副本中读取)，不能保证线程安全
	volatile可以保证有序性，即可以防止指令重排。
		应用：单利模式DCL(双检查锁定),可以在mInstance变量添加volatile来保证new Instance();不发生指令重排
